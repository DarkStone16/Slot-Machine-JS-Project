Machine à sous

Instructions


Bienvenue sur notre machine à sous.
- Cliquez sur le bouton jouer, pour lancer la partie.
- Vous disposez d'une mise initiale de 20 crédits, chaque parties coûtent 1 crédit.
- Si vous avez 3 symboles identiques, vous gagnez 2 crédits.
- Alors que si vous avez tout les symboles identiques, vous gagnez 5 crédits.
